# HGMAIB
## Requirments
```python
python = 3.8 
pytorch = 1.11+cu113 
numpy = 1.22.4 
pandas = 2.0.3
scipy = 1.10.1
scikit-learn = 1.3.2
```
## How to use

Run the `preprocess.py` script to preprocess the raw datasets and construct the heterogeneous network

Run the `train_search.py` script to search for the optimal adaptive meta-graph. 

Run the `train.py` script to  train the HGMAIB model, and obtain prediction results along with evaluation metrics.
